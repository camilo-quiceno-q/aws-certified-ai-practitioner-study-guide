Welcome! This repository contains a comprehensive study guide for the **AWS Certified AI Practitioner (AIF-C01)** certification. 

This guide is designed for professionals preparing to validate their basic knowledge of artificial intelligence, machine learning, and generative AI technologies on the AWS cloud.

## 📋 Exam Structure & Domains / Estructura y Dominios del Examen

*   **Format:** 65 questions (50 scored, 15 experimental) / 65 preguntas.
*   **Time / Tiempo:** 90 minutes / 90 minutos.
*   **Passing Score / Puntaje mínimo:** 700 / 1000.
*   **Domains / Dominios:**
    1.  Foundations of AI and ML (20%) / Fundamentos de IA y ML
    2.  Foundations of Generative AI (24%) / Fundamentos de IA Generativa
    3.  Applications of Foundation Models (28%) / Aplicaciones de Modelos Fundacionales
    4.  Guidelines for Responsible AI (14%) / Pautas para una IA responsable
    5.  Security, Compliance, and Governance (14%) / Seguridad, cumplimiento y gobernanza
