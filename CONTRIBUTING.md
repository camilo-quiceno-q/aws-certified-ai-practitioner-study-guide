# Contributing to the AWS Certified AI Practitioner Study Guide

First off, thank you for considering contributing to this study guide! It is contributions like yours that keep the guide up-to-date and useful for the community.

## How Can I Contribute?

### Reporting Issues / Typos
If you find a typo, spelling error, or an outdated AWS service definition, feel free to open an issue or submit a Pull Request directly.

### Updating Content
AWS exams and services change rapidly. If you see that a service detail has changed or a new domain has been introduced to the AIF-C01 blueprint, please update the respective `study-guide-es.md` or `study-guide-en.md` file.

## Pull Request Process

1. Fork the repository and create your branch from `main`.
2. Commit your changes with clear description.
3. Make sure to update the guide in **both languages** if the change is a technical correction.
4. Submit a Pull Request.

Thank you for your help!
